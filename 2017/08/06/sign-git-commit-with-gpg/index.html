Sign Git Commit with GPG

2017-08-06
  • git
  • gpg

GPG, Gnu Privacy Guard, is a popular and wide used encryption tool compatible to PGP (Pretty Good Privacy) tool. By using widely-accepted asymmetric encryption methods, GPG becomes an handy utility for signing digital information with a personal or organizational credential or securing private or sensitive message with high level encryption.

This post is to integrate GPG with Git for more reliable and controllable source code management.

Sign Git commits and tags

When commit.gpgsign configuration option is toggled on for repository or globally, the -S is optional for git-commit sub-command.
